# Maave AI API

AI-powered maternal health assistant that explains EPDS (Edinburgh Postnatal
Depression Scale) screening results in warm, simple language via the Gemini API.

Features


Generates warm, supportive explanations of EPDS screening results
Never diagnoses, never alters the score or risk category — Gemini is
constrained by prompt rules to explain only
Lightweight FastAPI backend, easy to deploy on Render or similar platforms
API key kept out of source control via environment variables

Disclaimer

This tool does not diagnose postpartum depression or any other medical
condition. It is intended to help explain screening results in plain
language and encourage users to seek support from a qualified healthcare
professional when appropriate. It is not a substitute for professional
medical advice.
